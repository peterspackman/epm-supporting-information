# Supporting Information

Supporting data for the EPM paper.

## Contents

- **cifs/** — Optimized crystal structures in CIF format for each compound and force field/method combination.
- **tensors/** — Computed elastic tensor components (in text format) for each compound and computational method.
- **pbe-phonons/** — Phonon frequencies (`*-freqs.txt`) and eigenvectors (`*-vecs.npy`) from PBE-D4 calculations.
- **elastic_tensors.xlsx** — Summary spreadsheet of all elastic tensor data.
